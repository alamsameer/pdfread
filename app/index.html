üìÑ JSON Output Structure
Complete document data including:
  • document_id - Unique identifier
  • filename - Original PDF name
  • metadata - Title, author, page count, etc.
  • pages[] - Array of page objects containing:
    • page_number
    • content.text - Full page text
    • content.paragraphs[] - Text split into paragraphs
    • images[] - Embedded images with paths
    • word_count - Words per page
